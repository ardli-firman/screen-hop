## Implementation Summary

Added window mover functionality to `src/browser_move/window_mover.py`:

### Constants Added (lines 15-20)
- `SWP_NOZORDER = 0x0004` - SetWindowPos flag to maintain Z order
- `SWP_NOACTIVATE = 0x0010` - SetWindowPos flag to not activate window
- `SW_RESTORE = 9` - ShowWindow command to restore window
- `SW_MAXIMIZE = 3` - ShowWindow command to maximize window
- `SW_SHOW = 5` - ShowWindow command to show window

### Functions Added

#### `move_window_to_monitor(hwnd: int, monitor: Any) -> bool` (lines 80-94)
- Checks if window is maximized using `win32gui.IsZoomed(hwnd)`
- If maximized, restores with `ShowWindow(hwnd, SW_RESTORE)`
- Moves window using `SetWindowPos(hwnd, 0, x, y, w, h, SWP_NOZORDER | SWP_NOACTIVATE)`
- Maximizes on target monitor with `ShowWindow(hwnd, SW_MAXIMIZE)`
- Returns True on success, False on exception

#### `move_browser_to_external(browser_type: str) -> bool` (lines 97-106)
- Convenience function that:
  1. Gets external monitors via `get_external_monitors()`
  2. Finds browser window via `find_browser_window(browser_type)`
  3. Moves window to first external monitor
